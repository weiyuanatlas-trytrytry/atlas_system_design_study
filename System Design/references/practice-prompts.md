# Practice Prompts

Use this reference when the user wants drills, homework, or a progression plan.

## Foundation Questions

- Design a URL shortener
- Design a rate limiter
- Design a pastebin service
- Design an API rate quota dashboard
- Design a file storage service

## Intermediate Questions

- Design Dropbox
- Design Instagram news feed
- Design YouTube upload and streaming
- Design WhatsApp
- Design Uber dispatch at high level

## Advanced Questions

- Design Twitter/X timeline
- Design Google Docs collaboration
- Design a global notification platform
- Design Stripe-style payment webhooks
- Design a metrics and alerting pipeline

## Skill Progression

### Week 1

- caching
- load balancers
- SQL vs NoSQL
- CAP and consistency basics

### Week 2

- queues and async processing
- sharding and replication
- API design
- data modeling

### Week 3

- messaging and realtime systems
- feed systems
- search systems
- observability and reliability

### Week 4

- full mock interviews
- timed answers
- tradeoff drills
- retrospective review

## Drill Formats

### 15-minute drill

Practice only clarification, requirements, estimates, and top-level architecture.

### 30-minute drill

Practice a full answer with one deep dive.

### Red-team drill

After finishing a design, challenge it with:

- 10x traffic spike
- regional outage
- hot partition
- abusive clients
- stricter consistency requirement
